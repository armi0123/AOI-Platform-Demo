# AOI Platform Demo

## 中文介紹

這是一個使用 C# 與 Visual Studio 開發中的模組化 AOI 軟體平台。

本專案主要用於練習工業 AOI 軟體架構設計，包含：

- PLC 通訊
- Log 管理
- Config 管理
- Inspection Workflow
- UserControl 模組化設計

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
## English Introduction

AOI Platform Demo is a modular industrial AOI software architecture project developed using C# and Visual Studio.

Current focus:

- PLC Communication
- Logging System
- Configuration Management
- Inspection Workflow
- Modular UserControl Architecture
